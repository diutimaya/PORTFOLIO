Research

Autonomous Tour Guide System for Tourists In Progress

Developing a smart tourism system that provides seamless indoor–outdoor navigation and context-aware information using multi-sensor fusion.

Objective

Build an intelligent system that detects user location automatically and delivers relevant information without manual interaction.

Problem

GPS fails indoors while BLE and IMU systems have limitations like signal noise and drift. No existing system provides seamless transition with contextual content delivery.

Approach

  • BLE-based proximity detection using RSSI
  • IMU-based Pedestrian Dead Reckoning (PDR)
  • Magnetic field sensing for indoor localization
  • Hybrid sensor fusion for improved accuracy

Key Findings

  • No single system solves indoor + outdoor transition
  • BLE suffers from signal instability
  • IMU systems accumulate drift over time
  • Sensor fusion is essential for real-world deployment

Impact

Enables scalable, real-time, and user-friendly smart tourism experiences with minimal user interaction.
